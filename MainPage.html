NAOS es un sistema operartivo para desarrolladores Web. Incorpora de manera predeterminada programas indispensable para un buen programador web. Desde los editores de texto avanzados más potentes; como Sublime Text y Visual Studio Code, editores de texto plano como Libre Office, listados de pluggins, buscador Brave integrado; hasta el apartado de entretenimiento, incorporando Spotify,Discord,etc...
